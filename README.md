# FMC (Fair Model-Based Clustering)

This repository contains the official implementation of **FCA**, the algorithm proposed in the paper:  
["Fair Model-Based Clustering"] by *Jinwon Park, Kunwoong Kim, Jihu Lee, and Yongdai Kim.*  
Published in **[AAAI 2026](https://aaai.org/conference/aaai/aaai-26/)**.
